## Prerequisites
 - Eclipse IDE. 
 - PostgreSQL.
 
## Get Started
 1. Clone the Project or Download ZIP project(unzip this). 
 2. Open Eclipse, `File/Import/Maven/Existing Maven Projects`, Choose the clone directory.
 3. Run Application `ApirestApplication`.
